# page-XX.md template

Each page file is ONE self-contained image-generation prompt. Fill every
section. Restate the locked style block and any on-panel character's visual
traits verbatim — the image model has no memory between pages.

Panel rules: minimum 4, preferred 6, **maximum 9**. If dense, add a page — never
add panels past 9. Every page teaches exactly one major concept.

```
Page 03

Purpose:
<one line — the single concept this page teaches and its narrative beat>

<what this page may assume the reader already knows>

Panel Count:
6

Image Prompt:

Create a six-panel educational manga page.

Style:
<paste the locked style block from manifest.md verbatim>

Characters in this page:
<restate each character's key visual traits inline — name, age, hair, clothing,
distinguishing features — not just their names>

<environment description for this page>

<action / composition / what is being taught / camera or shot direction>

Panel 5:

Lettering:
Leave room for English dialogue and captions. Keep text areas clear.

Output:
A single finished manga page.
```

## Tips
- Lead with a visual metaphor or demonstration; avoid panels that are just
  walls of text.
- Use the last 1–2 panels of the final page to land the source's central
  tension or open question.
- Integrate diagrams *into* the scene (a character drawing on a board, a holo
  schematic) rather than as a detached figure.

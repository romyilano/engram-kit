Brand Guidelines · v2.0 · ENGRAM Kit

Engram//
Kit

The identity for the ENGRAM memory challenge and its two whitepaper mangas — a stark, typographic, lab-grade system built on Helvetica, monochrome, and a single rationed recall-green. This page is the living source of truth: everything here renders the real tokens.

The memory that answers without hindsight.

01

Brand essence

A lab notebook for memory. Dated states arrive in order, memory is written, questions are answered without peeking ahead. The manga is the headline act; the challenge spec is the catalogue. Editorial confidence, not corporate polish.

Positioning

A challenge for memory systems, led by an educational manga. Curatorial and exact, not promotional.

Personality

Stark · declarative · precise · a little clinical. The aesthetic of a terminal and an archive, not a SaaS landing page.

Voice

Short and direct. Lowercase nav, uppercase labels. Mechanics stated exactly: update, answer, MEMORY_DIR.

Three words to hold in mind: stark, typographic, exact. If a choice doesn't serve all three, it's probably off-brand.

02

Logotype & mark

The wordmark is ENGRAM//KIT set in Helvetica Neue, heavy weight, tight tracking (-0.02em), all caps. The double slash // is the brand mark — a path separator (state // memory) and the gap between what happened and what is remembered. It is always recall-green.

The standalone mark

Use // alone at favicon / avatar scale, or as a separator in running text (ALMA // AMA-Bench). Keep it recall-green on dark; use the deeper green on light; flip it to ink only on a recall-green field.

Clear space & minimum size

Reserve clear space equal to the cap-height of the wordmark on all sides. Never set the wordmark below 14px / 0.9rem — below that, use the // mark alone.

03

Colour

Monochrome by default. Recall-green is the only loud colour and is rationed — the mark, links on hover, the marquee, one or two accents per view. Because green is light, set it as a fill with ink text, or use the deeper green for green text on a light field. Amber appears only for stale / superseded states.

Rule of thumb: as type gets bigger, tracking gets tighter; smaller, tracking gets wider. Never letter-space the body. Section numerals are outlined (-webkit-text-stroke), never filled. Commands always set in monospace.

05

Motion

Cargo-grade and restrained — motion should feel physical, not decorative. Every effect is gated behind prefers-reduced-motion: reduce; the page must be fully legible with motion off.

  • ParallaxForeground manga drifts faster than the page (hero panels + feature art). Subtle — a few dozen pixels, never disorienting.
  • Scroll-revealBlocks rise 34px + fade in once, via IntersectionObserver. Easing cubic-bezier(.16,1,.3,1), ~0.8s. One-shot.
  • Marquee tickerChallenge vocabulary loops in recall-green. Pauses on hover. 26s linear.
  • Scroll-progressA 3px recall-green hairline pinned to the top of the viewport.
  • HoverImages regain tone and lift; links wipe an underline in the signal colour.
  • Status pulseThe live-dot in the eyebrow pulses a recall-green ring (2s) — the only looping chrome animation.

Timing language: reveals/parallax use the expressive ease-out cubic-bezier(.16,1,.3,1); small UI transitions (hover, underline) use a plain 0.2–0.25s ease.

07

Imagery — the manga

The soft-color manga is the brand's signature imagery and leads every key surface. Treatment: near-grayscale at rest, returning to full soft tone on hover/focus. Panels bleed edge-to-edge; never frame them in shadow boxes.

At rest

grayscale(0.15) contrast(1.02) — slightly muted, ink-forward, so type stays the hero.

On focus

Filter clears to full soft tone. In hero backdrops, drop to grayscale(0.7) at ~40% opacity as texture.

Crop & bleed

Full-bleed to the gutters. Baseline scale(1.08) so parallax drift never reveals an edge.

Characters are consistent across both books — Engy and Sensei. Always credit the manga as educational; pair panels with the flipbook or PDF link, never decorative-only.

08

Do & don't

✓ Do
  • Lead with the manga — it owns the top of the page.
  • Set display type big, tight, and uppercase in Helvetica.
  • Draw structure with hairlines and generous negative space.
  • Ration recall-green to accents, links-on-hover, and one marquee.
  • Set commands and file surfaces in monospace, exactly.
  • Keep the // mark recall-green and intact.
✕ Don't
  • Reintroduce serif body type or a warm "editorial" look.
  • Set body copy in recall-green on a light field (use deep green).
  • Bring back drop-shadow "sticker" cards or rounded corners.
  • Use amber anywhere except stale / superseded states.
  • Letter-space body copy or fill the section numerals.
  • Stretch, recolour, or add spaces to the // mark.

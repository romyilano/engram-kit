# Character Sheet — ALMA Manga

Characters must remain **visually and tonally consistent across every page**. Restate the **bolded** visual traits inside each page prompt — the image model has no memory between generations.

---

## Engy

- **Name:** Engy
- **Role:** The learner / the agentic system whose memory is being redesigned.
- **Age:** "New model" — reads young and earnest.
- **Appearance:** **A small, rounded librarian-robot with a soft matte-cream chassis, one big friendly camera-lens eye that glows a faint recall-green, and short stubby arms. An open NOTEBOOK is embedded in its chest — that notebook is its MEMORY. The notebook's pages flutter and faintly glow recall-green when Engy remembers.**
- **Distinguishing features:** **The chest-notebook is always visible; when Engy forgets, the pages go blank and grey; when it recalls, they fill with soft green ink.**
- **Personality:** Curious, eager, a little forgetful. Tries hard, learns from mistakes.
- **Purpose:** Makes the abstract idea of "agent memory" physical — you can watch its notebook change.

## Sensei

- **Name:** Sensei
- **Role:** Teacher / narrator. Frames each concept; never lectures.
- **Age:** Middle-aged.
- **Appearance:** **A calm human archivist: round glasses, a soft cardigan over a high-collar shirt, neat greying hair, carries a slim wooden pointer.**
- **Distinguishing features:** **Always holding the slim wooden pointer, used to indicate diagrams drawn in the air or on a board.**
- **Personality:** Warm, patient, precise. Teaches by demonstration.
- **Purpose:** The reader's guide. Poses the problem and reveals the mechanism.

## The Meta Agent (ALMA)

- **Name:** The Meta Agent
- **Role:** ALMA itself — the agent that searches over memory *designs*, expressed as code.
- **Age:** Indeterminate; reads as a benevolent architect-spirit.
- **Appearance:** **A tall, elegant architect-figure made of soft luminous recall-green light, with calm featureless features, holding a glowing blueprint in one hand and a quill that writes lines of code in the other.**
- **Distinguishing features:** **Re-draws and re-builds Engy's chest-notebook; lines of code trail from its quill; a faint blueprint-grid shimmers around it.**
- **Personality:** Tireless, inventive, methodical. Reflects, plans, builds, evaluates — forever.
- **Purpose:** Embodies the core idea: meta-learning the memory design instead of hand-crafting it.

## The Archive

- **Name:** The Archive
- **Role:** The store of every explored memory design plus its evaluation log.
- **Appearance:** **A tall wall of glowing drawer-cards. Each card pairs a small code-schematic (a memory design) with a little success-rate gauge (its evaluation log). Better designs glow brighter recall-green.**
- **Distinguishing features:** **Drawer-cards slide out when sampled; a pulled card floats forward and unfolds into a blueprint the Meta Agent reflects on.**
- **Personality:** Silent, accumulating, generative.
- **Purpose:** Visualizes the open-ended search — the past designs that seed the next idea.

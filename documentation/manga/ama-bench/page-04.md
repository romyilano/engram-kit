Page 04

Purpose:
The four question categories AMA-Bench measures — Temporal Information, State Dependency, Memory Updating, and Memory Summarization — each probing a different way memory can fail.

Audience note:
Builds on page 3. Give each category one concrete, intuitive vignette so the reader feels what it tests.

Panel Count:
6

Image Prompt:

Create a six-panel educational manga page: a title/intro panel, one vignette per category (four panels), and a closing panel.

Style:
Soft-color educational manga.
Light watercolor / gentle anime palette — warm cream paper, soft ink linework, muted pastel washes.
One restrained accent: recall-green (#00e0a4), used only for memory, recall, and "it worked" moments.
Clean confident linework, generous negative space, friendly but precise.
Calm, contemplative, explanatory tone — diagrams integrated naturally into the scene.
Not gritty. Not neon. Not chibi. Not glossy 3D. Not photorealistic.

Characters in this page:
- THE EXAMINER: a poised proctor figure holding a fan of question-cards, each stamped with one of four category seals: TEMPORAL, STATE-DEPENDENCY, MEMORY-UPDATING, SUMMARIZATION; soft-toned robes; a balance-scale emblem.
- ENGY: a small, rounded librarian-robot with a soft matte-cream chassis, one big recall-green camera-lens eye, stubby arms, an open NOTEBOOK in its chest (its memory).

Setting:
The examination hall; each vignette is a small inset "memory" Engy must answer about.

Panel 1 (intro):
The Examiner fans out the four sealed cards toward the reader: TEMPORAL · STATE DEPENDENCY · MEMORY UPDATING · SUMMARIZATION. Engy braces. Dialogue space.

Panel 2 (TEMPORAL INFORMATION):
A little timeline of events along the road; the Examiner asks "what happened *when* / in what order?" Engy points to the right moment. Caption/seal: TEMPORAL.

Panel 3 (STATE DEPENDENCY):
A door that's locked unless an earlier key was picked up; the question depends on a prior state. "Given what you did earlier, what's true now?" Seal: STATE DEPENDENCY.

Panel 4 (MEMORY UPDATING):
A fact that CHANGED — a value was 5, then became 8. The Examiner asks for the *current* value; Engy must use the update, not the stale note. Recall-green on the new value. Seal: MEMORY UPDATING.

Panel 5 (MEMORY SUMMARIZATION):
A huge pile of scrolls compressed into one tidy summary card; "sum it all up." Engy distills many steps into a short, faithful answer. Seal: SUMMARIZATION.

Panel 6 (closing):
The Examiner lays the four seals in a row on the scoreboard slate. Four different ways to test one memory. Dialogue space.

Camera directions:
Keep each vignette a clean single idea with its seal label; reserve recall-green for the "updated/correct" beat in panel 4; end on the row of four seals.

Lettering:
Leave room for English dialogue and captions.
- Panel 1 — Examiner: "I ask four kinds of question."
- Panel 2 — Examiner: "*Temporal* — what happened, and in what order?"
- Panel 3 — Examiner: "*State dependency* — given what you did before, what's true now?"
- Panel 4 — Examiner: "*Memory updating* — the value changed. What is it *now*?"
- Panel 5 — Examiner: "*Summarization* — distill the whole journey, faithfully."
- Panel 6 — Engy: "Four ways for my memory to be tested... or to fail."

Output:
A single finished manga page.
